## 1. 课程概述

### 1.1 Data Science 工作流程

```
收集数据 → 提出模型 → 模型选择 → 做出决策 → 测试决策性能
```

### 1.2 数据科学所需技能

- **Programming（编程）** - 自动化重复任务，快速响应
- **Probability Theory（概率论）** - 量化不确定性
- **Statistics（统计学）** - 从数据推断模型
- **Machine Learning（机器学习）** - 预测与分类
- **Simulations（模拟）** - 近似复杂计算

### 1.3 概率论 vs 统计学

| 领域 | 问题 | 方向 |
|------|------|------|
| **概率论** | 给定数据生成过程，结果有什么性质？ | 过程 → 结果 |
| **统计学** | 给定结果，关于生成过程能说什么？ | 结果 → 过程 |

**示例：**
- 概率论：已知硬币正面概率为0.6，预测1000次抛掷中正面次数
- 统计学：观察到1000次抛掷中600次正面，推断硬币是否公平

### 1.4 常用软件中的条件概率应用

1. **天气预报应用** - 根据历史数据和当前条件计算降雨概率
2. **信用评分系统** - 根据财务历史计算贷款违约概率

## 2. 概率论基础

### 2.1 基本定义

**随机实验（Random Experiment）：** 即使以相同方式重复，也可能产生不同结果的实验。

**样本空间（Sample Space）S：** 所有可能结果的集合。

**事件（Event）：** 样本空间的子集。

**概率函数 P：** 满足以下条件的函数：
1. P(E) ≥ 0，对任意事件 E
2. P(S) = 1
3. 对互斥事件 E₁, E₂, ...，P(∪E) = ΣP(Eᵢ)

### 2.2 概率计算

**古典概率：**
$$P(E) = \frac{|E|}{|S|} = \frac{\text{有利结果数}}{\text{总结果数}}$$

**示例：** 掷骰子两次，求点数和为3的倍数的概率

解：
- 样本空间大小：6 × 6 = 36
- 满足条件的结果：12个
- P(A) = 12/36 = 1/3

### 2.3 集合运算与概率

| 运算 | 符号 | 含义 | 概率公式 |
|------|------|------|----------|
| 并集 | A ∪ B | A或B发生 | P(A∪B) = P(A) + P(B) - P(A∩B) |
| 交集 | A ∩ B | A和B都发生 | P(A∩B) |
| 补集 | Aᶜ | A不发生 | P(Aᶜ) = 1 - P(A) |

**互斥事件（Mutually Exclusive）：** A ∩ B = ∅，P(A∩B) = 0

### 2.4 计数原理

**乘法原理：** 若任务分n步完成，第i步有mᵢ种方法，则总方法数为 m₁ × m₂ × ... × mₙ

**示例：** 三餐选择问题
- 主菜：3种（虾、烤牛肉、鱼）
- 主食：2种（意面、米饭）
- 甜点：4种（冰淇淋、果冻、苹果派、桃派）

若A = 选冰淇淋，B = 选鱼，则 A∩B 的结果数 = 1 × 2 × 1 = 2

## 3. 随机变量与期望

### 3.1 随机变量定义

**随机变量（Random Variable）：** 从样本空间到实数的映射。

**示例：** 掷硬币3次，X = 正面次数
- 样本空间：{HHH, HHT, HTH, HTT, THH, THT, TTH, TTT}
- X(HHT) = 2, X(HTT) = 1

### 3.2 离散随机变量

**概率质量函数（PMF）：** f(x) = P(X = x)

性质：
1. f(x) ≥ 0
2. Σₓ f(x) = 1

### 3.3 期望（Expectation）

**定义：**
$$E[X] = \sum_{x} x \cdot P(X = x)$$

**线性性质：**
$$E\left[\sum_{i} C_i X_i\right] = \sum_{i} C_i E[X_i]$$

其中 Cᵢ 为常数，无需假设 Xᵢ 独立。

### 3.4 方差（Variance）

**定义：**
$$Var(X) = E[(X - E[X])^2] = E[X^2] - (E[X])^2$$

**性质：**
$$Var(aX + b) = a^2 Var(X)$$

## 4. 离散分布

### 4.1 Bernoulli 分布

**参数：** p（成功概率）

**PMF：**
- P(X = 1) = p
- P(X = 0) = 1 - p

**示例：** 抛硬币、游戏输赢

**期望与方差：**
- E[X] = p
- Var(X) = p(1-p)

### 4.2 Binomial 分布

**参数：** n（试验次数）, p（成功概率）

**定义：** n次独立实验中成功的次数

**PMF：**
$$P(X = k) = \binom{n}{k} p^k (1-p)^{n-k}$$

**期望与方差：**
- E[X] = np
- Var(X) = np(1-p)

### 4.3 其他离散分布

| 分布 | 参数 | 期望 | 方差 | 应用场景 |
|------|------|------|------|----------|
| Bernoulli | p | p | p(1-p) | 单次试验 |
| Binomial | n, p | np | np(1-p) | n次独立试验 |
| Poisson | λ | λ | λ | 稀有事件计数 |
| Geometric | p | 1/p | (1-p)/p² | 首次成功所需试验数 |

## 5. 连续分布

### 5.1 连续随机变量

**定义：** 可在区间内取任意值的随机变量。

**特点：** P(X = x) = 0（单点概率为0，但不是不可能事件）

### 5.2 概率密度函数（PDF）

**定义：** f(x) 满足：
1. f(x) ≥ 0
2. ∫∞^∞ f(x)dx = 1
3. P(a ≤ X ≤ b) = ∫ₐ f(x)dx

**注意：** f(x) 不是概率，而是概率密度。

### 5.3 正态分布（Normal Distribution）

**参数：** μ（均值）, σ²（方差）

**PDF：**
$$f(x|\mu,\sigma) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

**记法：** X ~ N(μ, σ²)

**性质：**
- 对称于 x = μ
- σ 越大，分布越平坦
- 68-95-99.7 规则

### 5.4 均匀分布（Uniform Distribution）

**参数：** a, b（区间端点）

**PDF：**
$$f(x) = \begin{cases} \frac{1}{b-a}, & a \leq x \leq b \\ 0, & \text{其他} \end{cases}$$

**记法：** X ~ Unif(a, b)

**期望与方差：**
- E[X] = (a+b)/2
- Var(X) = (b-a)²/12

## 6. 条件概率

### 6.1 定义

**条件概率：**
$$P(A|B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0$$

**含义：** 已知B发生的条件下，A发生的概率。

### 6.2 乘法公式

$$P(A \cap B) = P(A|B)P(B) = P(B|A)P(A)$$

### 6.3 独立事件

**定义：** A与B独立 ⟺ P(A|B) = P(A) ⟺ P(AB) = P(A)P(B)

**性质：**
- 若A与B独立，则A与Bᶜ也独立
- 多个事件相互独立需满足所有子集独立

### 6.4 全概率公式

若B₁, B₂, ..., Bₙ 是样本空间的一个划分，则：
$$P(A) = \sum_{i=1}^{n} P(A|B_i)P(B_i)$$

### 6.5 贝叶斯公式

$$P(B_i|A) = \frac{P(A|B_i)P(B_i)}{\sum_{j=1}^{n} P(A|B_j)P(B_j)}$$

### 6.6 生日问题

**问题：** n个人中至少两人生日相同的概率

**解：**
$$P(\text{至少两人相同}) = 1 - P(\text{全部不同}) = 1 - \frac{365 \times 364 \times ... \times (365-n+1)}{365^n}$$

**问题：** n个人中至少一人与"我"生日相同的概率

**解：**
$$P(A) = 1 - P(A^c) = 1 - \left(\frac{364}{365}\right)^n$$

当 n = 253 时，P(A) > 0.5

## 7. 最大似然估计

### 7.1 基本概念

**似然函数（Likelihood Function）：**
$$L(\theta|x_1, ..., x_n) = \prod_{i=1}^{n} f(x_i|\theta)$$

**最大似然估计（MLE）：** 选择使似然函数最大的参数值。

### 7.2 MLE 求解步骤

1. 写出似然函数 L(θ)
2. 取对数得对数似然函数 ℓ(θ) = ln L(θ)
3. 求导：dℓ/dθ = 0
4. 解方程得 MLE

### 7.3 常见分布的 MLE

| 分布 | 参数 | MLE |
|------|------|-----|
| Bernoulli | p | $\hat{p} = \frac{\sum x_i}{n}$ |
| Binomial | p | $\hat{p} = \frac{\sum x_i}{n}$ |
| Normal | μ | $\hat{\mu} = \bar{x}$ |
| Normal | σ² | $\hat{\sigma}^2 = \frac{1}{n}\sum(x_i-\bar{x})^2$ |
| Poisson | λ | $\hat{\lambda} = \bar{x}$ |

### 7.4 示例

**问题：** 抛硬币4次，得到HHHT，求正面概率的MLE

**解：**
- 似然函数：L(p) = p³(1-p)
- 对数似然：ℓ(p) = 3ln(p) + ln(1-p)
- 求导：dℓ/dp = 3/p - 1/(1-p) = 0
- 解得：$\hat{p} = 3/4$

## 8. 回归分析

### 8.1 线性回归模型

**模型：**
$$Y \sim N(\beta_0 + \beta_1 X, \sigma^2)$$

**含义：** 给定X，Y服从正态分布，均值为β₀ + β₁X，方差为σ²

### 8.2 参数估计（MLE）

**斜率估计：**
$$\hat{\beta}_1 = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2}$$

**截距估计：**
$$\hat{\beta}_0 = \bar{y} - \hat{\beta}_1\bar{x}$$

### 8.3 回归分析步骤

```
C. 提出模型 → B. 参数估计(MLE) → A. 模型检验(残差分析)
```

### 8.4 示例

**数据：**
| 年龄(X) | 18 | 19 | 20 | 20 | 21 | 22 |
|--------|----|----|----|----|----|----|
| 体重(Y) | 115 | 117 | 128 | 125 | 151 | 162 |

**计算：**
- $\bar{x} = 20$, $\bar{y} = 133$
- $\hat{\beta}_1 = 12.8$
- $\hat{\beta}_0 = -123$

**回归方程：** Y = 12.8X - 123

## 9. 置信区间

### 9.1 基本概念

**置信区间：** 基于样本数据构造的参数估计区间。

**置信水平：** 区间包含真实参数的概率（如95%）。

### 9.2 正态总体均值的置信区间

**已知σ²：**
$$\bar{x} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$$

**未知σ²：**
$$\bar{x} \pm t_{\alpha/2, n-1} \cdot \frac{s}{\sqrt{n}}$$

### 9.3 中心极限定理

**定理：** 当样本量n足够大时，样本均值的抽样分布近似正态分布：
$$\bar{X} \sim N\left(\mu, \frac{\sigma^2}{n}\right)$$

**应用条件：** n ≥ 30 通常认为足够大

## 10. 历年真题解析

### 10.1 题型分布

| 题型 | 分值 | 题数 |
|------|------|------|
| 选择题 | 20分 | 10题 |
| 计算题 | 80分 | 6-7题 |
| **总计** | **100分** | **7-9题** |

### 10.2 选择题考点

1. **概率论 vs 统计学** - 区分两者的问题方向
2. **MLE目标** - 找到使观测数据最可能的参数值
3. **PDF性质** - 连续变量单点概率为0
4. **互斥事件** - P(A∪B) = P(A) + P(B)
5. **独立事件** - P(A∩B) = P(A)P(B)
6. **条件概率** - P(A|B)的定义
7. **中心极限定理** - 样本均值分布趋近正态
8. **MLE计算** - 如硬币偏置估计
9. **回归模型参数** - σ²的含义
10. **正态分布性质** - σ变化对形状的影响

### 10.3 计算题考点

#### 考点1：生日问题

**解题思路：**
1. 定义样本空间
2. 计算补集概率
3. P(A) = 1 - P(Aᶜ)

#### 考点2：条件概率与计数

**解题思路：**
1. 列出所有可能结果
2. 找出满足条件的结果
3. 计算概率

#### 考点3：蒙特卡洛积分

**步骤：**
1. 在积分区间内均匀采样
2. 计算函数值
3. 取平均并乘以区间长度

**原理：** 大数定律

#### 考点4：MLE计算

**步骤：**
1. 写出似然函数
2. 取对数
3. 求导并解方程

#### 考点5：线性回归

**步骤：**
1. 计算$\bar{x}$, $\bar{y}$
2. 代入公式计算$\hat{\beta}_1$, $\hat{\beta}_0$
3. 写出回归方程

## 11. 公式汇总

### 11.1 概率公式

| 公式 | 表达式 |
|------|--------|
| 条件概率 | $P(A|B) = \frac{P(A \cap B)}{P(B)}$ |
| 乘法公式 | $P(A \cap B) = P(A|B)P(B)$ |
| 全概率公式 | $P(A) = \sum P(A|B_i)P(B_i)$ |
| 贝叶斯公式 | $P(B_i|A) = \frac{P(A|B_i)P(B_i)}{\sum P(A|B_j)P(B_j)}$ |
| 独立性 | $P(A \cap B) = P(A)P(B)$ |

### 11.2 期望与方差

| 公式 | 表达式 |
|------|--------|
| 期望定义 | $E[X] = \sum x \cdot P(X=x)$ |
| 期望线性 | $E[\sum C_i X_i] = \sum C_i E[X_i]$ |
| 方差定义 | $Var(X) = E[X^2] - (E[X])^2$ |
| 方差性质 | $Var(aX+b) = a^2 Var(X)$ |

### 11.3 常见分布

| 分布 | PMF/PDF | 期望 | 方差 |
|------|---------|------|------|
| Bernoulli | $p^x(1-p)^{1-x}$ | p | p(1-p) |
| Binomial | $\binom{n}{k}p^k(1-p)^{n-k}$ | np | np(1-p) |
| Normal | $\frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$ | μ | σ² |
| Uniform | $\frac{1}{b-a}$ | $\frac{a+b}{2}$ | $\frac{(b-a)^2}{12}$ |

### 11.4 回归与估计

| 公式 | 表达式 |
|------|--------|
| 斜率估计 | $\hat{\beta}_1 = \frac{\sum(x_i-\bar{x})(y_i-\bar{y})}{\sum(x_i-\bar{x})^2}$ |
| 截距估计 | $\hat{\beta}_0 = \bar{y} - \hat{\beta}_1\bar{x}$ |
| 似然函数 | $L(\theta) = \prod f(x_i|\theta)$ |
| 对数似然 | $\ell(\theta) = \sum \ln f(x_i|\theta)$ |
